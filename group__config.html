#define METAL_COMPAT_MODE

#include <metal/config/config.hpp>

Description

When this preprocessor switch is defined, certain language features that are known to cause problems to some compilers are avoided.

Warning
Defining this preprocessor switch may lead to longer compilation times.
Note
This preprocessor switch is currently defined by default for all versions of Microsoft Visual Studio.
